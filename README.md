# Calculadora de IMC

Uma calculadora simples de Índice de Massa Corporal (IMC) feita em JavaScript puro (vanilla JS), sem frameworks ou bibliotecas externas.

O usuário preenche altura e peso em um formulário HTML, e a aplicação calcula o IMC automaticamente, exibindo o resultado junto com a classificação correspondente (abaixo do peso, peso normal, sobrepeso, obesidade grau 1, 2 ou 3).

## 📋 Funcionalidades

- Cálculo do IMC a partir de altura e peso informados
- Validação de campos vazios ou inválidos
- Classificação automática do resultado em diferentes níveis
- Exibição dinâmica do resultado na página, sem recarregar (usando `preventDefault`)

## 🛠️ Tecnologias utilizadas

- HTML
- JavaScript (manipulação de DOM, eventos)
